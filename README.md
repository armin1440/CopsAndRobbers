# CopsAndRobbers
In this game you enter a length and width for the playground and then you enter the number of police stations and police officers corresponding to each of them.
Then the chase starts.The cops who have seen or have been informed about the robber's location move toward robber's location.
A cop sees the robber when the robber is in a 5x5 square which the cop is in the center of it ( starting from one, position(3,3)).
When a cop sees the robber , tells the cops in the same police station where the robber is.
The game ends when the robber is trapped ( surrounded by cops) or a cop catches the robber.
